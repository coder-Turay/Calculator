# 🧮 Python GUI Calculator

A simple calculator application built with Python and Tkinter that performs basic mathematical calculations through a graphical user interface.

## ✨ Features

* Clean and user-friendly interface
* Supports basic arithmetic operations:

  * Addition (`+`)
  * Subtraction (`-`)
  * Multiplication (`*`)
  * Division (`/`)
* Displays results instantly
* Clear button to reset the input
* Built using Python's built-in `eval()` function for expression evaluation

## 🛠️ Technologies Used

* Python 3
* Tkinter

## 🚀 How to Run

1. Clone or download this repository.
2. Run the Python script.

```bash
python main.py
```

3. Use the calculator interface to perform calculations.
